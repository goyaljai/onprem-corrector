# Grievance & Disclosure Policy

Agents must inform customers of their rights and the escalation path when relevant.

## Required Disclosures
- Grievance: on any complaint, the agent must mention the nodal officer / grievance redressal path | keywords: nodal officer, grievance, redressal, escalate
- Purpose: the agent must state the reason for the call | keywords: regarding, calling about, this call is about
